## 📌 About the Project  

**EasyStay** is a modern **MERN-based hostel management system** designed to simplify the process of **room allocation, complaint management, and student services**.  

It allows students to **request rooms, track approvals, and raise maintenance complaints**, while helping admins efficiently manage hostel operations.

## ✨ Features  

### 🛏️ Room Request System  
- 📩 Apply for hostel room allotment  
- ⏳ Track request status (Pending / Approved / Rejected)  
- 🏢 View available rooms  

### 🛠️ Complaint Management  
- 📝 Raise complaints (water, electricity, furniture, etc.)  
- 🔄 Track complaint status  
- 📊 Manage active complaints  

### 👩‍🎓 Student Dashboard  
- 👋 Personalized welcome dashboard  
- 📈 Quick stats overview  
- ⚡ Easy navigation  

### 🔐 Authentication  
- 🔑 Secure login/signup  
- 👤 Role-based access (Student/Admin)
